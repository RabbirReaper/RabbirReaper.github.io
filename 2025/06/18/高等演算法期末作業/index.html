隨機生成邊

這邊是我認為最困難的地方，要隨機路段又要確保都有連通，我最後採取了先生成最小生成樹，然後再隨機生成一些線段，我在思考應該使用Prim演算法還是Kruskal演算法，最後我選擇了我幾乎沒寫過的Kruskal演算法。

隨機生成垃圾在線段上

這邊使用了一些數學，根據隨機的t來計算垃圾的相對位置

P = A + t * (B - A) //2D 向量插值公式
int x = (int)Math.round(e.start.x + t * (e.end.x - e.start.x));
int y = (int)Math.round(e.start.y + t * (e.end.y - e.start.y));
