甚麼是 EEG

腦電圖（Electroencephalogram, EEG）是一種測量大腦電活動的醫學檢查技術。它透過在頭皮上放置電極來記錄大腦神經元產生的電訊號，這些訊號會以波形的方式呈現，讓醫生和研究人員能夠了解大腦的功能狀態。

EEG 的基本原理

大腦中的神經元在活動時會產生微弱的電流，這些電流會傳導到頭皮表面。EEG 設備能夠檢測到這些極其微弱的電訊號（通常只有幾微伏特），並將其放大記錄下來。不同的腦波頻率反映了不同的大腦狀態：

  • Delta 波（0.5-4 Hz）：深度睡眠狀態
  • Theta 波（4-8 Hz）：淺睡眠或深度放鬆狀態
  • Alpha 波（8-13 Hz）：清醒但放鬆的狀態
  • Beta 波（13-30 Hz）：清醒且專注的狀態
  • Gamma 波（30-100 Hz）：高度專注或認知處理狀態

常見使用情況

EEG 在醫學和研究領域有廣泛的應用：

醫學診斷方面：

  • 癲癇診斷和監測是最常見的用途，能夠檢測異常的腦波活動
  • 睡眠障礙研究，包括睡眠呼吸中止症、失眠等
  • 腦部疾病診斷，如腦炎、腦瘤、中風後的大腦功能評估
  • 昏迷患者的意識狀態評估
  • 認知功能障礙的評估，如失智症

研究和應用方面：

  • 神經科學研究，了解大腦的認知機制
  • 腦機介面（Brain-Computer Interface, BCI）技術
  • 神經回饋治療，幫助改善注意力不足過動症（ADHD）
  • 冥想和放鬆訓練的效果監測
  • 麻醉深度監控

資料觀察重點

1. 🔍 資料品質檢查 - 第一優先級

基本檔案資訊觀察

信號品質指標

觀察重點：

實際觀察步驟：

  1. 查看統計表格中的 Std Dev (uV) 欄位
  2. 尋找明顯異常的數值（特別大或特別小）
  3. 標記品質可疑的頻道，後續分析時特別留意

2. 📈 時域分析觀察重點

時間序列波形觀察

主要觀察目標：

A. 正常EEG特徵

B. 異常模式識別

觀察技巧：

  1. 分段檢視：不要只看開頭30秒，要多檢查不同時段
  2. 對比觀察：比較左右對稱頻道的差異
  3. 關注轉換：注意清醒/睡眠狀態轉換期
  4. 標記時段：記錄可疑時間點供進一步分析

3. 🌊 頻域分析觀察重點

頻帶功率分布解讀

A. 正常頻帶功率範圍

B. 異常頻帶模式

功率譜密度觀察

觀察重點：

  1. 峰值位置：是否出現不正常的頻率峰值
  2. 能量分布：各頻帶間的相對比例
  3. 頻道差異：不同腦區的頻譜特徵差異
  4. 週期性成分：是否有規律的頻率模式

4. 🚨 癲癇檢測指標觀察

異常評分系統理解

異常評分 (Anomaly Score) 解讀：

多維度異常指標

觀察重點組合：

A. 高頻活動 (High Frequency Power)

  • 正常值：通常<0.001
  • 異常值：>0.01需要關注
  • 意義：高頻活動增加可能預示癲癇發作

B. 尖波計數 (Spike Count)

  • 正常值：每分鐘<5個尖波
  • 異常值：每分鐘>20個尖波
  • 意義：尖波是癲癇的典型電生理標誌

C. 能量變異度 (Energy Variability)

  • 正常值：0.1-0.5
  • 異常值：>1.0
  • 意義：能量劇烈變化可能表示病理活動

5. 📊 視覺化圖表解讀技巧

時間序列圖觀察

關鍵觀察點：

熱力圖 (Heatmap) 解讀

顏色模式理解：

  • 深色/冷色：較低的功率值
  • 亮色/暖色：較高的功率值
  • 異常模式：某個頻道某個頻帶特別突出

觀察策略：

  1. 橫向比較：同一頻道不同頻帶的分布
  2. 縱向比較：同一頻帶不同頻道的差異
  3. 區域聚集：相鄰頻道是否有相似模式

功率譜圖解讀

關鍵觀察：

  • 主要峰值：在哪些頻率有明顯峰值
  • 平滑度：譜線是否平滑或有尖銳峰值
  • 斜率：高頻段的衰減是否正常

12. 📚 總結：關鍵觀察原則

核心觀察策略

  1. 系統性：按照固定流程逐項檢查
  2. 對比性：多維度、多時段比較
  3. 整體性：結合多個指標綜合判斷
  4. 動態性：關注變化趨勢而非靜態值
  5. 謹慎性：疑似異常需要專業確認
