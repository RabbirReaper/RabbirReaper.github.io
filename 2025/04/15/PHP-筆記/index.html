根據你提供的作業內容和常考題型，以下是幾個可能的PHP上機考試題目及參考答案，我會著重在PHP的處理邏輯，而不是CSS樣式：

猜題1: 學生成績處理系統

題目要求:
建立一個學生成績處理系統，能夠:

  1. 輸入5名學生的姓名和成績
  2. 計算平均分數、最高分、最低分
  3. 顯示及格(60分以上)和不及格的學生人數與名單
  4. 按照分數從高到低排序

參考答案:

猜題2: 網站會員註冊與登入系統

題目要求:
建立一個簡易會員系統，包含:

  1. 註冊頁面: 收集用戶名、密碼和確認密碼
  2. 透過 SESSION 模擬註冊和登入功能
  3. 登入成功後顯示用戶資訊
  4. 提供登出功能

參考答案:

猜題3: 改良的BMI計算器

題目要求:
根據作業中的BMI計算器，加入以下功能:

  1. 能計算多人的BMI (例如一組5人)
  2. 顯示所有人的BMI數值和體重狀態
  3. 計算所有人的BMI平均值
  4. 額外計算體脂率 (公式: 男性 = 1.2 × BMI + 0.23 × 年齡 - 5.4，女性 = 1.2 × BMI + 0.23 × 年齡 - 16.2)

參考答案:

猜題4: 多國貨幣轉換計算器

題目要求:
根據你的貨幣轉換作業，擴展功能:

  1. 支援多種貨幣之間的轉換 (美元、歐元、日圓、人民幣和新台幣)
  2. 顯示轉換歷史記錄 (使用SESSION儲存)
  3. 提供清除歷史記錄的功能

參考答案:

猜題5: 書籍管理系統 (與作業類似)

題目要求:
擴展作業中的書籍管理系統，加入以下功能:

  1. 新增書籍表單 (書名、作者、分類)
  2. 儲存新增的書籍 (使用SESSION)
  3. 按分類顯示書籍列表
  4. 搜尋書籍功能 (依書名或作者)

參考答案:

猜題6: 購物車與訂單系統

題目要求:
創建一個簡易的購物車與訂單系統，包含:

  1. 商品列表頁面 (顯示商品名稱、價格、加入購物車按鈕)
  2. 購物車頁面 (顯示已選商品、數量、小計、總計)
  3. 結帳頁面 (輸入姓名、地址、電話)
  4. 確認訂單頁面 (顯示訂單資訊與商品明細)
